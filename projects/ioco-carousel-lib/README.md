# iOCO Carousel Component Library

## Install

```bash
npm install ioco-carousel-lib
```

### Sub Module Import

```typescript
import { IocoCarouselLibModule } from "ioco-carousel-lib";

@NgModule({
  imports: [
    // ...

    IocoCarouselLibModule,
  ],
})
```

### Usage

```html
<ioco-carousel-lib configOptions="options">
  <h1>Title</h1>
</ioco-carousel-lib>
```
